Can VisKo Render a 3D Binary Float Array in a format that Mozilla Firefox can Display?

PREFIX viskoV: <http://trust.utep.edu/visko/ontology/visko-view-v3.owl#>
PREFIX viskoO: <http://trust.utep.edu/visko/ontology/visko-operator-v3.owl#>
PREFIX viskoS: <http://trust.utep.edu/visko/ontology/visko-service-v3.owl#>
PREFIX owlsService: <http://www.daml.org/services/owl-s/1.2/Service.owl#>
PREFIX owlsProcess: <http://www.daml.org/services/owl-s/1.2/Process.owl#>
PREFIX owl: <http://www.w3.org/2002/07/owl#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX xsd: <http://www.w3.org/2001/XMLSchema#>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>

ASK
			
WHERE
{{
?viewer viskoO:partOfViewerSet <http://rio.cs.utep.edu/ciserver/ciprojects/visko/mozilla-firefox1.owl#mozilla-firefox1> .
?viewer viskoO:operatesOn ?format .
<http://rio.cs.utep.edu/ciserver/ciprojects/formats/BINARYFLOATARRAY.owl#BINARYFLOATARRAY> viskoO:canBeTransformedToTransitive ?format . 
} UNION {

?viewer viskoO:partOfViewerSet <http://rio.cs.utep.edu/ciserver/ciprojects/visko/mozilla-firefox1.owl#mozilla-firefox1> .
?viewer viskoO:operatesOn ?format .
<http://rio.cs.utep.edu/ciserver/ciprojects/formats/BINARYFLOATARRAY.owl#BINARYFLOATARRAY> viskoO:canBeTransformedTo ?format .
} UNION {

?viewer viskoO:partOfViewerSet <http://rio.cs.utep.edu/ciserver/ciprojects/visko/mozilla-firefox1.owl#mozilla-firefox1> .
?viewer viskoO:operatesOn <http://rio.cs.utep.edu/ciserver/ciprojects/formats/BINARYFLOATARRAY.owl#BINARYFLOATARRAY> .
}}
